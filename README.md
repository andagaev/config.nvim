# My NeoVim config

## Installation

### Clone

```
git clone https://github.com/andagaev/config.nvim.git ~/.config/nvim
```

### Install all LSP servers

```
:MasonInstallAll
```

### Install additional binaries

- [ripgrep](https://github.com/BurntSushi/ripgrep)
- [fd](https://github.com/sharkdp/fd)
